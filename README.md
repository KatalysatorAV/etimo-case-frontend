## Develop locally

1.  **Clone repo**

    ```shell
    git clone https://github.com/KatalysatorAV/etimo-case-frontend.git
    ```

2.  **Navigate into directory**

    ```shell
    cd etimo-case-frontend
    ```

3.  **Install dependencies**

    ```shell
    npm i
    ```

4.  **Start development server**
    ```shell
    npm start
    ```
